# Projeto backend com C#

### Extensões no VSCODE

- C#
- C# Extensions
- C# Dev Kit
- NuGet Gallery
- .NET Install Tool

---

#### Iniciando projeto
```c#
dotnet new webapi
```

#### Instalando pacote para conexão com Banco de dados
```c#
Microsoft.EntityFrameworkCore
```

#### Instalando pacote para conexão com SQLite
```c#
Microsoft.EntityFrameworkCore.Sqlite
```
